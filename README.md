# 🐛 Insect Detector - تشخیص حشرات قرنطینه‌ای

اپلیکیشن موبایل اندروید برای تشخیص حشرات قرنطینه‌ای ایران با استفاده از هوش مصنوعی.

## 🎯 کاربرد
این اپلیکیشن به بازرسان قرنطینه و کارشناسان کشاورزی کمک می‌کند تا با گرفتن عکس از حشرات، به سرعت نوع آن‌ها را تشخیص دهند.

## 🧠 فناوری‌های استفاده شده
- **Flutter:** فریم‌ورک توسعه موبایل
- **TensorFlow Lite:** موتور اجرای مدل هوش مصنوعی روی موبایل
- **YOLOv8:** الگوریتم آموزش مدل تشخیص حشرات
- **تعداد کلاس‌ها:** ۳۹ نوع حشره قرنطینه‌ای ایران

## 📱 نحوه استفاده
1. اپلیکیشن را روی گوشی اندروید نصب کنید
2. یکی از دو دکمه را بزنید:
   - **دوربین:** برای گرفتن عکس مستقیم از حشره
   - **گالری:** برای انتخاب عکس از تصاویر ذخیره شده
3. نام حشره و میزان اطمینان مدل نمایش داده می‌شود

## ⚙️ ویژگی‌ها
- ✅ کاملاً آفلاین (بدون نیاز به اینترنت)
- ✅ پشتیبانی از دوربین و گالری
- ✅ رابط کاربری ساده و فارسی
- ✅ نمایش درصد اطمینان تشخیص

## 📊 مدل هوش مصنوعی
- **فرمت مدل:** TensorFlow Lite (`.tflite`)
- **نوع مدل:** Classification (دسته‌بندی تصویر)
- **سایز ورودی:** 224×224 پیکسل
- **دقت:** Float32
